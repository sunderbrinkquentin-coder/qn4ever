# 💕 Unsere Liebe - Relationship App

Eine wunderschöne Web-App zur Pflege eurer Beziehung mit modernem Design in Türkis/Grün.

## 🚀 Features

✨ **Login-System** - Mit persönlichen Vornamen
📅 **Interaktiver Kalender** - Mit Countdown zum nächsten Besuch
💌 **Liebesnachrichten** - Mit 8 kreativen Vorschlägen
📊 **Wöchentliches Feedback** - 4 wichtige Fragen
🎨 **Modernes Design** - Türkis/Grün, responsive

## 📂 Dateien

- **`unsere-liebe-premium.html`** - Die Hauptdatei (öffnen im Browser)
- **`unsere-liebe-app.html`** - Alternative Version

## 🌐 Online öffnen

### Option 1: GitHub Pages
Datei wird direkt aus GitHub angezeigt:
```
https://github.com/sunderbrinkquentin-coder/qn4ever
```

### Option 2: Netlify Deployment
1. Gehe zu https://app.netlify.com
2. Melde dich mit GitHub an
3. Verbinde dein Repository
4. Wähle `unsere-liebe-premium.html` als Publish-Datei
5. **Fertig!** 🎉

### Option 3: Lokal öffnen
Öffne einfach die HTML-Datei im Browser:
```
C:\Users\sunde\source\repos\qn4ever\unsere-liebe-premium.html
```

## 💾 Datenspeicherung

Alle Daten werden lokal im Browser gespeichert (localStorage):
- Login-Informationen
- Termine & Besuche
- Nachrichten
- Feedback

**Wichtig:** Löscht den Browser-Cache nicht, sonst gehen die Daten verloren!

## 🎨 Design

- **Hauptfarbe:** Türkis (#1b7a6b)
- **Akzentfarbe:** Mintgrün (#5ecfc9)
- **Responsive:** Funktioniert auf Handy, Tablet & Desktop

## 🔐 Sicherheit

Diese App speichert Daten nur lokal im Browser. Keine Daten werden an Server übertragen.
